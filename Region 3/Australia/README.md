# Australia (VK)

The regulator in Australia is the Australian Communications and Media Authority ([ACMA](https://www.acma.gov.au/)). It controls license approvals and callsign allocations.

As of 2024-02-19, Australia moved to a class licensing system, where "a qualified person with a suitable call sign may operate an amateur station" .. in other words, you need to hold a qualification and a callsign in order to transmit on amateur radio frequencies.

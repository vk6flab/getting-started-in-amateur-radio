# Germany (DL)
In Germany, amateur radio is regulated through the Amateurfunkgesetz (AfuG) with the details such as allowed modes and powers laid out in the Amateurfunkverordnung (AfuV). The authority in charge is the Bundesnetzagentur (BNetzA). In order to obtain a license, you need a certificate (Amateurfunkzeugnis) either issued through the BNetzA itself after passing several exams, or issued by another CEPT member state.

In Germany, 3 different license classes are defined: Class N, Class E and Class A. These correspond to CEPT agreements [ECC Report 89](https://docdb.cept.org/download/409), [ERC Report 32](https://docdb.cept.org/download/2065) and [HAREC TR-61/02](https://docdb.cept.org/download/2569) respectively.

* With a Class N license, you are allowed to use a transmit power of 10W ERP/EIRP (depending on band), the 10m, 2m and 70cm bands, and use repeaters as well as club stations.
* The Class E license extends this to up to 100W PEP (depending on band), the 160m, 80m, 15m, 23cm and all bands above and including 13cm. Additionally to the Class N, you are allowed to train trainees using your own callsign.
* The Class A license allows up to 750W PEP (depending on band) and the use of all bands available for amateur radio in Germany, i.e., additionally to the ones from Class N and E: 2.2 km, 630 m, 60 m, 40 m, 30 m, 20 m, 17 m, 12 m, 6 m. Furthermore, you are allowed to use remote stations.

For the certificate allowing you to apply for a Class N license, you need to pass exams in the following areas: 
* Operation (Betriebliche Kenntnisse, B)
* Legal foundations (Kenntnisse von Vorschriften, V)
* Basic technical knowledge (Technische Kenntnisse, N)
These exams can be taken in a single session at one of the branch offices of the BNetzA (registration required).

For a certificate allowing you to apply for a Class E license, you need all of the above for Class N plus Advanced technical knowledge (Technische Kenntnisse, E). You can either take this exam in a different sitting after having passed the exams for Class N, i.e., upgrade your license, or you can take all 4 exams in once.

For a certificate allowing you to apply for a Class A license, you need all 4 exams above plus Expert technical knowledge (Technische Kenntnisse, A). Again, you can either upgrade from a lower license or take all 5 exams in one sitting.

# Amateur radio clubs in Germany
The biggest amateur radio club in Germany is the [Deutscher Amateurradioclub (DARC)](https://www.darc.de/home/). Albeit a non-government organisation, the DARC actively shapes the regulations of amateur radio in Germany through the [Runder Tisch Amateurfunk (RTA)](http://www.runder-tisch-amateurfunk.de/). For example, the catalog of questions for the exams taken at the BNetzA including all the figures was created by a committee of volunteers inside the DARC. However, in contrast to other countries, the DARC is not allowed to conduct certificate exams as listed above, as this is considered a sovereign task in German legislation. The DARC operates a website with comprehensive material for the preperation for the amateur radio exams at the BNetzA. It can be found at [50ohm.de](https://50ohm.de/) (German language only).

# Callsigns
Germany has been assigned the prefixes DA through DR, but DA through DO are the prefixes for regular stations. As of 2026-07-02, DN9... is used for license Class N stations, DO1-9... and DA6... for Class E and DA1... through DM9... for Class A. Details can be found in the [Rufzeichenplan](https://www.bundesnetzagentur.de/DE/Fachthemen/Telekommunikation/Frequenzen/SpezielleAnwendungen/Amateurfunk/DL_Vfg_AFu/Rufzeichenplan_10_24_Auszug_aus_Vfg_15_2025.pdf?__blob=publicationFile&v=3).

# Roaming
In general, it is possible to operate an amateur radio license from another CEPT member state without the need to apply for a German license by prefixing the foreign callsign with DL/ for Class A-equivalent licenses and DO/ for Class E-equivalent licenses, as long as the German regulations are respected. Radio amateur from non-CEPT states need to apply for a guest license at the BNetzA. More details in English language can be found [here.](https://www.darc.de/der-club/referate/ausland/english-version/#c519)

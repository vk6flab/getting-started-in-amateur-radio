# Getting Started in Amateur Radio

Getting Started in the hobby of Amateur Radio can be a challenge if you don't
know anyone in the community. This collection of documents and resources aims
to help you find your way.
* [Start Here](start-here.md)


## Structure

There is a generic component to every amateur radio journey, mainly to do
with how the hobby and the community surrounding it is organised and
structured.

For each country, there are specific requirements and considerations that
are likely to change from time-to-time, mostly enforced by the various
regulatory bodies and officials controlling radio spectrum. Each country
is represented by a folder with resources pertinent to that particular
entity.

You'll note that the world is split into three regions. While this might look
arbitrary, it actually reflects the structure adopted by the International
Telecommunications Union (ITU) and the International Amateur Radio Union (IARU).


## Updates

This is likely to be updated from time-to-time. Changes and patches are
